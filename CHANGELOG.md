### Added

- Initial implementation: seven WCAG 2.2 AA checks for Google Docs (alt text, heading
  structure, link text, color contrast, document title, tables, drawings/charts).
- Sidebar UI with plain-language explanations, Locate, and one-click fixes for alt text
  and link text.
- Host-agnostic checks engine with full unit-test coverage; Docs adapter; esbuild + clasp
  build pipeline; CI.
